## 💻 Geliştirme Ortamı

Projeyi yerel bilgisayarınızda çalıştırmak için:  

```bash
# Depoyu klonlayın
git clone https://github.com/Berat-365/fluxoDev.git

# Klasöre girin
cd fluxoDev

# Gerekli paketleri yükleyin
npm install

# Geliştirme sunucusunu başlatın
npm run dev
